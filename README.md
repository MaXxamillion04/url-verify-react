To run this locally, 

first: clone the repo to somewhere local.

On Linux:

  cd into repo
  
  sudo npm install
  
  sudo npm start



you may need to [install npm first](https://linuxize.com/post/how-to-install-node-js-on-ubuntu-18.04/)

open browser to [http://localhost:3000](http://localhost:3000)
